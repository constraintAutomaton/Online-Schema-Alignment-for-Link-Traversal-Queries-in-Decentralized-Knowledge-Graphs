\begin{abstract}
The integration of heterogeneous \acrlongpl{kg} remains challenging, particularly in decentralized environments where discovery queries execute over distributed, unindexed sources requiring alignment rules for unified querying.
This demo paper presents an online schema alignment approach integrated into a \acrlong{ltqp} engine using a web-based interface.
A social media scenario demonstrates querying across networks with varying vocabularies.
Source code, library, and command-line interface are provided for integration into other applications.
\end{abstract}
