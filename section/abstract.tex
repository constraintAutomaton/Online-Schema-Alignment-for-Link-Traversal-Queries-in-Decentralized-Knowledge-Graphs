\begin{abstract}
% context
Decentralized \acrlongpl{kg} querying enables integrating distributed data without centralization, but is highly sensitive to vocabulary heterogeneity.
% need
Expecting query issuers to anticipate all vocabulary mismatches is unrealistic, especially when alignment rules are local, scoped or discovered at runtime.
% object
We present an \emph{online schema alignment} approach for \acrfull{ltqp} that discovers, scopes, and applies alignment rules dynamically during query execution while preserving traversal behavior.
% task of this paper
This demo paper demonstrates the approach on a decentralized social-media scenario through a web interface built on a Comunica-based \acrshort{ltqp} engine.
Source code, a CLI, and a reusable library are publicly available.
% conclusion / perspective
The demonstration shows that online schema alignment recovers complete query results with low overhead, providing a practical foundation for web-scale reasoning in \acrshort{ltqp} systems.
\end{abstract}
